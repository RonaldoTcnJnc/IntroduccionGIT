Resumen del proyecto

Esta página web presenta una demostración de trabajo con Git y GitHub. Se diseñó un sitio moderno que explica el flujo de versiones, los commits y la integración con un repositorio remoto.

Diseño visual

Se aplicaron efectos de vidrio, gradientes y una paleta oscura para dar un estilo profesional y elegante.

Control de versiones

El proyecto incluye commits descriptivos, un reset a un punto anterior y posteriores modificaciones para demostrar el manejo de versiones.

Integración

Se conectó el repositorio local al remoto en GitHub y se subieron los cambios a la rama master.
